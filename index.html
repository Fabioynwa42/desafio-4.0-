Sobre este projeto

Este dashboard foi desenvolvido como produto tecnológico do Desafio 4.0 da disciplina de Lógica e Matemática Discreta (UNDB, 2026.1). O objetivo é demonstrar como conceitos de lógica proposicional, tabelas-verdade e análise de dados podem ser aplicados a problemas socioeconômicos reais.

Os dados são baseados na pesquisa Raio X do Investidor Brasileiro da Anbima (2025) e o modelo de avaliação de risco utiliza como referência o Problem Gambling Severity Index (PGSI), adaptado de forma simplificada para fins educacionais.

Aviso educacional: Este sistema é um protótipo com fins acadêmicos, desenvolvido para ilustrar a aplicação de lógica proposicional à identificação de comportamentos de risco. Não substitui diagnóstico clínico. Em caso de suspeita de problema com apostas, consulte um profissional de saúde.
